# 🍽️ Fork & Flame – Restaurant Menu UI/UX Design

A modern restaurant menu mobile application designed as part of the **CodeVedX UI/UX Internship Program**.

## 📖 Project Overview

Fork & Flame is a user-friendly restaurant menu experience focused on helping customers easily browse dishes, explore categorized menus, view detailed food information, and proceed seamlessly through the ordering process.

## ✨ Key Features

* Premium dark-themed UI with elegant gold accents
* Categorized food sections for easy navigation
* Rich food imagery and detailed dish descriptions
* Search and promotional offers integration
* Interactive cart and checkout experience
* Mobile-first, intuitive user experience

## 📱 Screens Designed

* Welcome / Splash Screen
* Home & Featured Menu Screen
* Food Browsing Screen
* Dish Details Screen
* Cart & Checkout Screen

## 🎯 Project Objectives Achieved

✔ Visually appealing and engaging interface

✔ Easy-to-use navigation and menu browsing

✔ Clear categorization of dishes

✔ Enhanced customer ordering experience
